== Description ==
AdSimple Vote ermöglicht es Fragen an die Leser von Beiträgen zu stellen und wertet die Antworten aus. 
Mit Hilfe eines Buttons im Editor kann man ganz leicht den Shortcode einbauen, nachdem man die Frage und 2 mögliche Antworten angegeben hat.

== Installation ==

1.  Laden Sie das Plugin in Ihr '/wp-content/plugins' Verzeichnis.
2. Aktivieren Sie das Plugin uber das 'Plugins' Menu in WordPress.
